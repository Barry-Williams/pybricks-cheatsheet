This function I have created to use the gyro sensor to turn instead of estimating by wheel turn

# Create function for gyroTurn

LEFT = -1 # Left negative

RIGHT = 1 # Right positive

def turn(direction, angle, speed=50):

    gyro.reset_angle(0)

    robot.stop()

    while abs(gyro.angle()) < angle:

        right_motor.run(speed * direction)

        left_motor.run(speed * direction * -1)

    right_motor.stop()

    left_motor.stop()

but these commands by themselves appear to do nothing. These commands return the value and can be useful in logic commands like if and while we can look at later but first lets see what these values are. There are 2 ways we can get python to show us information.

print(robot.distance())            # This prints out to VS Code in the output window

ev3.screen.print(robot.distance()) # This prints out on the EV3 screen

The problem now is this will only print the value once and not update it. We need to use a loop to get it to print continuously.

while True:

    print(robot.distance())          

    ev3.screen.print(robot.distance())

# Initially this will just display 0 but if you move the push the robot or move the wheels the value will change

# This will run forever so you can stop it by pressing stop in VS Code or the button under the screen on the robot
